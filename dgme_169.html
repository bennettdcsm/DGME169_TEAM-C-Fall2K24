DGME 169



Web and Mobile Design III:

Mobile First Design

Students utilize HTML & CSS skills to create mobile first and responsive websites. Students will learn the fundamentals of responsive web design (RWD) that include media queries, fluid grids and flexible images. Mobile first design will be an essential theme of the course, as well as User Experience and Interactive Design strategies. Students will develop a functional prototype. Software: Adobe Creative Suite®
